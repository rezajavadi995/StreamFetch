# StreamFetch

Terminal-based media downloader for Termux, powered by yt-dlp and FFmpeg.

[فارسی (Persian)](README_FA.md)

---

## Features

- Download audio as MP3  
- Download video as MP4  
- Supports YouTube and SoundCloud  
- Download progress display  
- Retry on failed downloads  
- FFmpeg audio conversion  
- Lightweight and simple CLI interface  

---

## Requirements

- Python  
- FFmpeg  
- yt-dlp  
- Termux (currently supported)

---

## Installation

```bash
git clone https://github.com/rezajavadi995/StreamFetch.git
cd StreamFetch
bash install.sh
```

Run:

```bash
downloader
```
