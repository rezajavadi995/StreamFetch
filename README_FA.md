# StreamFetch

دانلودر مدیا برای ترموکس، مبتنی بر yt-dlp و FFmpeg.

[English](README.md)

---

## قابلیت‌ها

- دانلود صوت با فرمت MP3  
- دانلود ویدیو با فرمت MP4  
- پشتیبانی از یوتیوب و ساندکلاد  
- نمایش وضعیت دانلود  
- تلاش مجدد هنگام خطا  
- تبدیل فایل با FFmpeg  
- رابط خط فرمان سبک و ساده  

---

## پیش‌نیازها

- Python  
- FFmpeg  
- yt-dlp  
- Termux (فعلاً)

---
